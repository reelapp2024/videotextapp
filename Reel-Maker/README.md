# Video Text Overlay Tool

A powerful React-based application for adding text overlays to videos with Excel data integration.

## Features
- Batch video processing with text overlays
- Excel data integration for dynamic text
- Multiple overlay configurations
- Audio extraction and mixing
- Auto-download and ZIP packaging
- Real-time preview

## Installation

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

1. Upload your videos
2. Upload audio files (optional)
3. Upload Excel file with text data
4. Configure text overlays and settings
5. Click "Start Processing"

Enjoy!
